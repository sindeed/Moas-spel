# Moas spel

En samlingssida för spel skapade av Moa.

Webbplatsen publiceras med GitHub Pages och kommer att finnas på:

<https://sindeed.github.io/Moas-spel/>

## Struktur

- `index.html` – portalens innehåll
- `styles.css` – design och responsiv layout
- `script.js` – små förbättringar i gränssnittet

Spelen läggs senare i egna mappar och länkas från startsidan.
